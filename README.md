# Тестовое задание для Клеверенс Софт
Репозиторий включает 3 тестовых задачи согласно техзаданию.

Язык исполнения C#, версия платформы была выбрана .NET 8.

Исходные коды в папке src.
Каждая задача в отдельном каталоге и с отдельным проектом.

**Задача 1** отражена в проекте CompDecomp.
В проекте реализована компрессия и декомпрессия строки.

**Задача 2** отражена в проекте StaticCountApp.
В проекте реализован класс-сервер с возможностью множественного чтения и эксклюзивной записи для только одного потока за раз.

**Задача 3** отражена в проекте LogFormatter.
В проекте реализовано распознавание формата из предоставленного перечня (2 варианта),
а также методы нормализации формата в формат по образцу.
Классы спроектированы таким образом, чтобы можно было добавлять другие форматы и отлаживать их нормализацию независимо.
